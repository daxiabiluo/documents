# 9.1 操作步骤

具体操作如下： 

1. 单击“添加”按钮； 
2. 在弹出的编辑框中添加不重复的js名称、选择执行方式和填写相应的参数； 
3. 单击选中需要运行的函数； 
4. 点击“Add”按钮，在弹出的“选择点”窗口中选择一个需要读取或者设置的点； 
5. 单击“插入”按钮，将新建的函数添加到js编辑框的光标后面； 
6. 可重复3,4,5的步骤进行多个函数的添加； 
7. 点击”确定“按钮完成js脚本的编辑。
